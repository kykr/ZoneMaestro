Perceptual Evaluation of 3D Indoor Scene Generation

Please evaluate 3D indoor scenes generated by different methods based on text instructions.

Evaluation Guidelines

You will be shown a series of cases. Each case consists of a text instruction describing a desired 3D indoor scene, and seven rendered scenes generated by different methods (labeled Method A–G, in randomized order).

For each case, please evaluate all seven methods along three dimensions. For each dimension, select all methods whose output satisfies the criterion:

  • Zone Assignment: Objects are placed in the correct functional zones as specified (e.g., "dining area", "lounging zone").
  • Relative Placement: Spatial relationships match the instruction (e.g., "near the wall", "central seating cluster", "aligned to the perimeter").
  • Object Combination: The specified object groups/combinations are present in the scene (e.g., "kitchen cabinets and appliances", "sofas and armchairs").
